WINDOWS SYSTEM REQUIREMENTS
- Microsoft Windows 95(R) or better, Windows NT (R)
- SVGA display (minimum High Color, 16 Bit; 640 x 480 pixel resolution) 
required.
- 16 MB of free RAM required to launch PowerPoint
- Double-speed CD-ROM drive required, quad-speed recommended.
- Soundblaster-compatible sound card required for products including sound
- Speakers or headphones recommended for products including sound
- PowerPoint application

WINDOWS 95/98 INSTALLATION AND STARTUP
1. Turn on your computer and start Windows.
2. Insert the Ahrens Multimedia Manager CD-ROM into your CD-ROM drive.
3. Launch Windows Explorer and locate CD-ROM drive and content on CD
4. Copy PowerPoint chapter lecture desired to hard drive. 
5. If you wish to make changes to the chapter lecture, click on Properties using 
Right Mouse. Uncheck the read only box, click apply and OK. 
4. To view PowerPoint lectures double click on lecture file to launch 
PowerPoint. 

---------------------------------------------------------
MACINTOSH SYSTEM REQUIREMENTS
- Macintosh PowerPC and iMac platform with MacOS 7.6.1 or higher.
- Color display (minimum thousands of colors), minimum 640 x 480 pixel 
resolution.
- 32 MB of free RAM required to use PowerPoint. 
- Double-speed CD-ROM drive required, quad speed recommended.
- Headphones recommended for products including sound
- PowerPoint application

MACINTOSH INSTALLATION AND STARTUP 
1. Insert the Ahrens Multimedia Manager CD-ROM into your CD-ROM drive. 
2. Double-click on the CD icon. 
3. To Launch PowerPoint files, Move selected Chapter Lecture File or Chapter 
Archive File to
hard drive
4. Double Click on PowerPoint file to launch file and PowerPoint application.
5. To make changes or additions to the PowerPoint file, open PowerPoint 
application first and open PowerPoint chapter file located on hard drive. Make 
changes and saved as desired.

MACINTOSH INSTALLATION NOTES: 
1) Turn Virtual Memory off. Having Virtual Memory on increases the likelihood of 
a system crash. 
2) Some users may need to turn off Modern Memory Manager in the memory control 
panel for proper 
operation. 
3) Make sure that 32-bit addressing is turned on.
